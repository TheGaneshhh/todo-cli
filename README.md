# todo-cli
A simple command-line To-Do List application built in Python.
This project helps users add, view, and mark tasks as done and remove task — all from the terminal.


## Features
- ➕ Add new tasks  
- 📋 View all tasks  
- ✔️ Mark tasks as done
- ❌ Remove task
- ❌ Quit anytime with `q`
